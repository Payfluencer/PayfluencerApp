# NodeJs Express Server

## Pre-requisites

- NodeJs (v20) LTS
- tsx (installed globally)
- pnpm@latest (package manager)

## Installation

### Install tsx (typescript executable)

```bash
npm install -g tsx
```

### Clone the repository

```bash
git clone git@github.com:
```

### Install dependencies

```bash
pnpm install
```

### Start the server

```bash
pnpm run dev
```
